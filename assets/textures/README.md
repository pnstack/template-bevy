# Textures

Place your game textures, sprites, and UI images here.

## Recommended formats:
- PNG (for sprites with transparency)
- WebP (for compressed textures)

## Organization suggestions:
- `player/` - Player sprites and animations
- `enemies/` - Enemy sprites
- `ui/` - User interface elements
- `backgrounds/` - Background images
- `tiles/` - Tilemap textures
